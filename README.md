Forest Fire Prediction using Machine Learning


-This project focuses on predicting forest fires using environmental factors like temperature, humidity, wind, and rainfall

-The main idea is to understand how these conditions affect fire risk and use machine learning for prediction

-I cleaned and preprocessed the dataset, explored patterns, and built a model to analyze fire occurrence

-Used Python along with Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, and Flask

-Worked with the Algerian forest fire dataset containing weather and fire-related data

-The model captures useful patterns and shows how ML can be applied to real-world problems
